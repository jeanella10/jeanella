GITHUB

Desarrollado por GitHub en colaboración con OpenAI, GitHub Copilot es un asistente de codificación que sugiere fragmentos de código y funciones mientras escribes. Utiliza un modelo de lenguaje para proporcionar sugerencias contextuales y completar líneas de código

CODEX

Codex es el modelo de lenguaje subyacente en GitHub Copilot y se especializa en la generación de código. Puede generar código en varios lenguajes a partir de descripciones en lenguaje natural.

REPLIT

Un entorno de desarrollo en línea que utiliza IA para ayudar en la codificación y proporcionar asistencia contextual.
